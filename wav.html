This package is for reading Wav headers (that is if you are lucky enough:). There is nothing to describe here: just open a binary stream with element type (unsigned-byte 8) as always and call wav:read-wav-header.

API

read-wav-header(stream)
Reads wav file header from stream. Returns two values: list of subchunks and size of audio data to read. Stream position is set to begining of audio data.
